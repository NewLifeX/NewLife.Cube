# NewLife.Cube.React

#### 介绍

魔方 React 前端皮肤包的前端源码，基于 React 18 + Ant Design 5 + UMI Max。

原始项目：[NewLife.CubeAntd](https://github.com/NewLifeX/NewLife.CubeAntd)

在线体验：https://antd.newlifex.com/

---

#### 本地开发

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev
```

#### 构建

```bash
pnpm build
```
