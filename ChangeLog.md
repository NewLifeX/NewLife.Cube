# ChangeLog

## v6.12.2026.0708 (2026-07-08)

### 认证与安全
- **外部认证集成**：支持第三方外部验证服务集成，访问规则支持响应码检测与动态封禁
- **IP 封禁策略升级**：支持 /24 和 /16 子网封禁策略，非 IPv4 地址跳过子网追踪安全加固
- **密码与 Session 安全增强**：增强密码重置流程安全性，修复 OAuth 回调后 Session ID 漂移问题
- **权限控制优化**：修复子类重载 Token 鉴权时的重复验证问题，确保无用户时正确设置数据权限

### LOV 值集系统
- **LOV 值集系统**：新增 List of Values 值集体系，支持值集管理、使用指南文档和功能更新

### 审计与日志
- **审计日志页面**：新增审计日志页面，支持日志查看和搜索功能
- **AI 日志诊断**：集成 AI 能力，支持日志分析与系统诊断

### 前端增强
- **API 层统一**：合并 cubeApi 与 usePageApi 为统一 composable useCubeApi
- **字段约束验证**：增加 controller 数据字段约束验证，模型绑定验证响应，前端验证提示
- **菜单增强**：菜单管理添加层级前缀标识和自适应高度布局，新增 visibleField 可见性字段映射配置
- **对话框/弹窗优化**：支持多行消息显示，新增命令式弹窗封装与组织规范
- **标签组件增强**：新增 TabsView 组件，修复标签点击跳转与溢出翻页，统一样式 Token
- **路由配置化**：路由命名风格可配置，视图组件匹配增强，登录页路径配置化

### SSO 与 OAuth
- **SSO 角色规则**：新增 SSO 角色规则属性，支持根据规则自动升级用户主角色
- **Token 管理**：实现 TokenService 和 UserBindingService，添加令牌滑动刷新阈值配置，支持 JWT 自动续期
- **OAuth 配置优化**：更新应用地址描述，优化重定向逻辑以支持提供者名称，增加自动重跳授权

### 头像系统
- **头像优化**：添加头像地址替换规则（公网→内网），内网失败后使用外网地址重试降级机制，SVG 文字头像自动生成，多格式头像查找与 Content-Type 动态返回

### 实体与后端
- **实体树控制器**：新增 WebApi 实体树控制器基类
- **全局实体链接字典**：支持全局实体类链接字典，外键跳转支持自动带 Area 前缀
- **数据库管理**：新增模型差异页面，WebApi 版 Debug 模式下默认开启跨域
- **区域注册优化**：重构区域注册基类，优化区域名称处理逻辑

### 工程与测试
- **E2E 测试**：72 项测试全部通过，修复登录竞态、重定向、session 注销、IP 封禁等场景
- **CI 精简**：移除前端构建与打包，仅保留后端，简化发布流程
- **依赖升级**：升级 XCode、NewLife.AI 等核心依赖

- **公共模块**：新增 `@cube/api-core`（统一 API 客户端、Token 管理、类型定义）和 `@cube/field-mapping`（字段元数据推断控件类型，20+ WidgetType）

  - `NewLife.Cube.ArcoVue` — Vue 3 + Arco Design Vue / 字节跳动（端口 5183）
  - `NewLife.Cube.Angular` — Angular 19 + NG-ZORRO / Ant Design（端口 5184）
  - `NewLife.Cube.Vuetify` — Vue 3 + Vuetify 3 Material Design（端口 5185）
  - `NewLife.Cube.Svelte` — SvelteKit 2 + Svelte 5 + Tailwind CSS 4（端口 5186）
  - `NewLife.Cube.TDesign` — Vue 3 + TDesign / 腾讯（端口 5187）
- **统一功能**：每套皮肤均支持动态菜单、字段元数据驱动列表/表单、CRUD、批量删除、CSV 导入导出、OAuth 登录、深色模式
- **NuGet 一键集成**：引用 NuGet 包即获完整前端，`app.UseXxx(env)` 或 IModule 自动注册
- **CI/CD**：更新 test/publish/publish-beta 三个 GitHub Actions 工作流，全部皮肤自动构建打包发布
