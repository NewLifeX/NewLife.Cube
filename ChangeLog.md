# ChangeLog

## v6.10.2026.0404 (2026-04-04)

### 富文本与 Markdown 编辑器
- **Quill + EasyMDE 集成**：支持富文本与 Markdown 双编辑器，可在实体字段上自由切换
- **HTML 源码模式**：富文本编辑器新增 HTML 源码查看与编辑按钮
- **Markdown 图片上传**：支持直接粘贴/拖拽图片并自动关联实体记录
- **上传安全增强**：富文本与附件上传增加权限细粒度控制与安全校验

### 表单功能增强
- **returnUrl 传递**：表单 Add/Edit 页面支持 returnUrl 参数的透传与提交后自动回跳
- **多值字段支持**：增强实体模型绑定，支持复选框等多值字段的绑定与提交
- **批量操作**：BatchAction 自动注入启用/禁用按钮，减少重复代码

### JSON 序列化
- **SystemJson 适配**：集成 System.Text.Json，支持通过 `Apply(options)` 统一配置序列化选项

### OAuth 与认证
- **部门 ID 类型调整**：OAuth 模型中部门 ID 与 ParentID 统一为 String 类型，增强兼容性
- **redirect_uri 兼容**：同时支持 `redirect_uri` 与 `redirect_url` 两种参数名，兼容第三方客户端
- **多层反代支持**：优化多层反向代理场景下协议头与真实 IP 的识别逻辑

### UI 优化
- **导航栏通知记录**：用户信息导航栏新增通知记录入口
- **配置页链接优化**：短信配置、邮件配置等系统配置页增加快捷导航链接
- **文件下载文件名**：优化文件下载时的文件名编码与设置逻辑

### Bug 修复
- **[fix]** 修复 EasyMDE 表格按钮因 Bootstrap 样式冲突导致的错位问题
- **[fix]** 修复 URL 重建时端口号为 0 的处理逻辑异常

### 文档与工程
- **文档全面汉化**：魔方使用手册新增多项技术运维章节，全面使用简体中文
- **升级代码生成器**：同步升级内置脚手架代码生成器版本

## v6.9.2026.0201 (2026-02-01)

- 支持多租户，重构角色/部门填充逻辑
- 魔方支持完整的数据权限控制以及多租户数据隔离，由XCode做底层支撑
- [feat] 增加短信邮件配置表，增加验证码记录表和通知记录表，支持多租户多渠道消息通知和验证码发送
- 支持邮箱验证码发送、绑定与重置功能，优化结构
- 支持手机号码登录与验证码验证
- [feat] OAuthConfig增加多租户支持
- 增加魔方用户手册和使用手册

### 改进优化
- 优化头像同步与下载流程，支持Token认证
- 增强SSO头像哈希校验与响应头支持
- 统一Token模型为IToken，提升多租户与认证兼容性
- 重构导航栏，优化多模块与移动端体验
- 导航栏模块菜单响应式优化及租户菜单合并
- 重构验证码服务，支持多渠道多租户短信/邮件
- 多租户架构优化与文档完善，支持多端接入
- 重构 GetRawUrl 返回类型为 UriInfo 并增强兼容性
- 地区数据初始化优化，提升性能与灵活性
- 删除TokenInfo替换为NewLife.Web.TokenModel
- 更新NewLife.Core核心库，TokenModel包含Scope
- 简单包装服务层返回
- 优化百分比字段显示精度与缩放逻辑
- 修正表单横向布局被-15px整体左移的问题
- 表单标签与描述超长时支持省略号显示
- 设置文件哈希相关响应头
- 遇到异常Host请求时，直接在异常里告知原始请求Url
- 验证号码有效只需要缓存，不需要读配置
- 手机验证码部分缓存前缀设为常量
- 建立cube版协作指令

### Bug修复
