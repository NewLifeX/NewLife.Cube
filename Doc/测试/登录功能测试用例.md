## 2. 密码登录（Challenge-Response）

### TC-PWD-001 正常登录（Challenge-Response 加密）

| 步骤 | 操作 | 期望结果 |
|------|------|---------|
| 1 | 打开登录页 | 页面正常加载 |
| 2 | 输入正确用户名和密码 | 表单可用 |
| 3 | 点击登录 | 浏览器自动调用 `GET /Auth/Challenge` 获取公钥 |
| 4 | （自动）前端用 RSA-OAEP/SHA-256 加密密码 | Network 中可见 `challengeId` 参数和密文密码 |
| 5 | 后端验证通过 | 跳转至首页，无报错 |

**适用皮肤**：全部 10 套

**适用皮肤**：全部 10 套（SPA 跳 `/forgot-password` 路由，MVC 切 Tab/Panel）

## 4. OAuth 第三方登录（returnUrl 传递）

### TC-OAUTH-001 OAuth 登录后跳转到指定 returnUrl

| 步骤 | 操作 | 期望结果 |
|------|------|---------|

## 5. 注册流程

### TC-REG-001 用户名密码注册（成功）

| 步骤 | 操作 | 期望结果 |
|------|------|---------|
| 1 | 打开注册页 | 页面正常加载，展示用户名/密码表单 |
| 2 | 输入可用用户名、邮箱、强密码、确认密码 | 表单通过前端校验 |
| 3 | 点击注册 | 调用 `POST /Auth/Register`，`registerCategory=0` |
| 4 | 后端创建用户并签发令牌 | 返回 `accessToken`/`refreshToken` |
| 5 | 前端自动登录并跳转首页 | 页面进入已登录态 |

### TC-REG-002 用户名重复

### TC-REG-003 手机验证码注册（成功）

| 步骤 | 操作 | 期望结果 |

| 4 | 后端成功创建并绑定账号 | 自动登录成功 |

## 6. 自动化测试（XUnit）

测试文件：`XUnitTest/AuthControllerTests.cs`

| 测试方法 | 覆盖点 |
|---------|-------|
| `RsaOaepEncryptDecrypt_RoundTrip` | RSA-OAEP 加解密往返正确性 |
| `RsaOaepEncrypt_NonDeterministic` | 每次加密结果不同（随机填充，防重放） |
| `PemSpkiExportImport_RoundTrip` | PEM/SPKI 公钥导出/导入等价验证 |
| `IsEmail_FormatValidation` (Theory) | 邮箱格式正则校验（5 组正反例） |
| `IsMobile_FormatValidation` (Theory) | 手机号格式正则校验（5 组正反例） |
| `WeakPassword_IsRejectedByRegex` (Theory) | 弱密码被正则过滤 |
| `StrongPassword_Passes` (Theory) | 强密码通过校验 |

运行命令：
```bash
dotnet test XUnitTest/XUnitTest.csproj --filter "FullyQualifiedName~AuthControllerTests" -v normal
```

新增建议测试文件：`XUnitTest/RegisterTests.cs`

| 测试方法 | 覆盖点 |
|---------|-------|
| `RegisterByPassword_Success` | 用户名密码注册成功并返回令牌 |
| `RegisterByPhoneCode_Success` | 手机验证码注册成功 |
| `RegisterByEmailCode_Success` | 邮箱验证码注册成功 |
| `Register_DuplicateUsername_Failed` | 用户名冲突校验 |
| `Register_WeakPassword_Failed` | 弱密码拒绝 |
| `Register_CodeInvalid_Failed` | 验证码错误 |
| `Register_Disabled_Failed` | AllowRegister=false 拒绝注册 |

新增接口回归测试文件：`XUnitTest/AuthRegisterApiTests.cs`

| 测试方法 | 覆盖点 |
|---------|-------|
| `OAuthPendingInfo_InvalidToken_ShouldFail` | 无效 OAuth token 返回失败 |
| `Register_PasswordNotMatch_ShouldFail` | 密码与确认密码不一致校验 |
| `Register_OAuthBind_WithoutToken_ShouldFail` | OAuthBind 缺少 token 校验 |
| `Register_PasswordMode_WithoutUsername_ShouldFail` | 密码注册缺少用户名校验 |

运行命令：
```bash
dotnet test XUnitTest/XUnitTest.csproj --filter "FullyQualifiedName~RegisterTests" -v normal
dotnet test XUnitTest/XUnitTest.csproj --filter "FullyQualifiedName~AuthRegisterApiTests" -v normal
```
